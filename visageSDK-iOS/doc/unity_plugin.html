Visage Tracker Unity Plugin (C# wrapper) project

The VisageTrackerUnityPlugin is a wrapper that exposes some of VisageTracker's functionality for use in managed programming languages such as C# and Java. A prebuilt version of the wrapper is provided in libs/libVisageTrackerUnityPlugin.a. It's use is demonstrated in VisageTrackerUnityDemo and FacialAnimationUnityDemo samples where a C# script is used to call functions exposed in this library. This project is designed to be integrated in other applications and cannot be run on its own.

Building the project

The files related to Visage Tracker Unity Plugin project are located in the Samples/iOS/VisageTrackerUnityPlugin sub-folder of the visage|SDK for iOS folder.

Prerequisites to building the sample project are :

  • XCode (version 6.x)
The VisageTrackerUnityPlugin can be built from the XCode project in the sample folder.

Implementation overview

The project consists of a XCode project and .mm and .h files in which a Visage Tracker Observer interface is implemented. The observer exposes some of Visage Tracker's properties such as translation, rotation and Action Units and wraps some of Visage Tracker's functions such as initiating and stopping tracking. More functionality can be exposed in the same manner. Refer to the API documentation for more information on how to access various properties and functionalities of the Visage Tracker.
