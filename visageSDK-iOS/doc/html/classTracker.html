Detailed Description

Class that implements the behaviour of tracking application.

This is the core class that shows how to use visage|SDK capabilities in Unity. It connects with visage|SDK through calls to native methods that are implemented in VisageTrackerUnityPlugin. It uses tracking data to transform objects that are attached to it in ControllableObjects list.
