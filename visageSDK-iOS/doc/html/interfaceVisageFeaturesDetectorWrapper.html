Detailed Description

Class that implements simple high-level Objective-C interface around visage|SDK Visage Features Detector functionallity.

This makes connecting the iOS app GUI with visage|SDK features detector functionaly easier as primary programming language used in iOS apps is Objective-C while primary programming language used in visage|SDK is C++. The class is a subclass of NSObject and it uses Objective-C++ for implementation details.
