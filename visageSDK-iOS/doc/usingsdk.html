Welcome to visage|SDK

visage|SDK integrates a comprehensive range of computer vision technologies related to human faces in an easy-to-use, fully documented Software Development Kit to support a wide area of applications. Main features are face and facial features detection and tracking including 3D head pose, gaze, eye closure, full 3D head model and more, as well as face analysis including gender and emotion estimation.

Documentation

visage|SDK documentation consists of this user manual and additional reference manuals for specific purposes (sample projects, tracker configuration etc.). These additional manuals are accessed through links in this document, the References section, or directly in the doc folder.

The menu on the left allows navigation through the documentation.

Getting started

visage|SDK includes sample projects with full source code demonstrating how to use API. It is recommended to start by running the samples, described in the Samples section. The API is given in the API section and contains full documentation of all classes organised by topic.
